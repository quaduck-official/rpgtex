# rpgtex

A collection of modular LaTeX packages for RPG typesetting.

## Packages

| File | Description |
|------|-------------|
| `rpg.sty` | Main package, loads all modules |
| `rpgcolors.sty` | Color schemes |
| `rpgcomment.sty` | Comment/margin note blocks |
| `rpgcore.sty` | Core macros and utilities |
| `rpgfonts.sty` | Font setup |
| `rpgheader.sty` | Page headers and footers |
| `rpglayout.sty` | Page layout and geometry |
| `rpgmonster.sty` | Monster stat blocks |
| `rpgoptions.clo` | Class options |
| `rpgreadaloud.sty` | Read-aloud text blocks |
| `rpgsections.sty` | Sectioning commands |
| `rpgsidebar.sty` | Sidebar environments |
| `rpgskilltree.sty` | Skill tree / ability diagrams |
| `rpgtable.sty` | Table environments |
| `rpgtoc.sty` | Table of contents styling |
| `rpgutility.sty` | Miscellaneous helpers |

## Usage

```latex
\usepackage{rpg}
```

See the `examples/` directory for sample documents.
